Lulu's Commands
===============

Here is lulu's command listing, which we have hopefully kept up-to-date for you:

```
<thing>++ - give thing some karma
<thing>-- - take away some of thing's karma
botsnack - give the bot a food
check domain <domainname> - returns whether a domain is available
lulu !! - Repeat the last command directed at lulu
lulu (redmine|show) me <issue-id> - Show the issue status
lulu <user> is a badass guitarist - assign a role to a user
lulu <user> is not a badass guitarist - remove a role from a user
lulu add <hours> hours to <issue-id> ["comments"] - Adds hours to the issue with the optional comments
lulu animate me <query> - The same thing as `image me`, except adds a few parameters to try to return an animated GIF instead.
lulu announce "<message>" - Sends a message to all lulu rooms.
lulu announce downtime complete for "<service>" - Syntactic sugar for announcing downtime completion
lulu announce downtime for "<service>" starting <timeframe> - Syntactic sugar for announcing downtime commencement
lulu assign <issue-id> to <user-first-name> ["notes"] - Assign the issue to the user (searches login or firstname)
lulu base64 encode|decode <query> - Base64 encode or decode <string>
lulu commit message - Displays a random commit message
lulu convert me <expression> to <units> - Convert expression to given units.
lulu decide "<option 1>" "<option 2>" "<option x>" - Randomly picks an option
lulu decide "<option 1>" "<option 2>" "<option x>" - Randomly picks an option
lulu decide <option1> <option2> <option3> - Randomly picks an option
lulu decide <option1> <option2> <option3> - Randomly picks an option
lulu die - End lulu process
lulu echo <text> - Reply back with <text>
lulu gem whois <gemname> - returns gem details if it exists
lulu help - Send a link to all of lulu's commands.
lulu help <query> - Displays all help commands that match <query>.
lulu how long since <event>? - Display the number of days since the event
lulu how many fucks has <user> given?
lulu how many fucks?
lulu image me <query> - The Original. Queries Google Images for <query> and returns a random top result.
lulu it's been <number> days since <event> - Set the day when the event happened
lulu karma <thing> - check thing's karma (if <thing> is omitted, show the top 5)
lulu karma best - show the top 5
lulu karma worst - show the bottom 5
lulu kitten me - A randomly selected kitten
lulu kitten me <w>x<h> - A kitten of the given size
lulu link me <issue-id> - Returns a link to the redmine issue
lulu lmgtfy <optional @username> <some query>
lulu map me <query> - Returns a map view of the area returned by `query`.
lulu marco - Reply with Polo
lulu math me <expression> - Calculate the given expression.
lulu mustache me <query> - Searches Google Images for the specified query and mustaches it.
lulu mustache me <url> - Adds a mustache to the specified URL.
lulu ping - Reply with pong
lulu pug me - Receive a pug
lulu set <issue-id> to <int>% ["comments"] - Updates an issue and sets the percent done
lulu show (my|user's) issues - Show your issues or another user's issues
lulu tell <recipients> <some message> - tell <recipients> <some message> next time they are present.
lulu the rules - Make sure lulu still knows the rules.
lulu throw a coin - Gives you heads or tails
lulu time - Reply with current time
lulu translate me <phrase> - Searches for a translation for the <phrase> and then prints that bad boy out.
lulu translate me from <source> into <target> <phrase> - Translates <phrase> from <source> into <target>. Both <source> and <target> are optional
lulu update <issue-id> with "<note>" - Adds a note to the issue
lulu uptime - Outputs bot uptime
lulu who is <user> - see what roles a user has
lulu xkcd <num> - XKCD comic <num>
lulu xkcd [latest]- The latest XKCD comic
lulu xkcd random - XKCD comic <num>
lulu youtube me <query> - Searches YouTube for the query and returns the video embed link.
listens for "who is " followed by the bot name
ship it - Display a motivation squirrel
```
